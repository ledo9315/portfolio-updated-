Zeitleiste

  • 16.05.2024 – Projektstart

    Projektstart und Planung. Wir haben die Anforderungen und Ziele des Projekts definiert und die Struktur der Webseite entworfen. Dabei haben wir uns für ein modernes und responsives Design entschieden, das auf CSS Grid und Flexbox basiert. Die Webseite soll eine klare und übersichtliche Struktur haben und auf allen Geräten optimal dargestellt werden.

  • 16.05.2024 – Typografie

    Auswahl und Integration von Schriftarten. Wir haben die Schriftarten Hanson und Montserrat als @font-face in die CSS-Datei eingebunden. Dabei wurden die Fonts in den Formaten .woff und .woff2 eingebunden. Font-display wurde auf swap gesetzt, um die Texte so schnell wie möglich anzuzeigen. 

    
  @font-face {
    font-family: "Hanson";
    src: url("../../fonts/Hanson-Bold.woff2") format("woff2"),
         url("../../fonts/Hanson-Bold.woff") format("woff");
    font-weight: bold;
    font-style: normal;
    font-display: swap;
  }
  
  @font-face {
    font-family: "Montserrat";
    src: url("../../fonts/Montserrat-Light.woff2") format("woff2"),
         url("../../fonts/Montserrat-Light.woff") format("woff");
    font-weight: 300;
    font-style: normal;
    font-display: swap;
  }
  
  @font-face {
    font-family: "Montserrat";
    src: url("../../fonts/Montserrat-Medium.woff2") format("woff2"),
         url("../../fonts/Montserrat-Medium.woff") format("woff");
    font-weight: 500;
    font-style: normal;
    font-display: swap;
  }
  
  @font-face {
    font-family: "Montserrat";
    src: url("../../fonts/Montserrat-Regular.woff2") format("woff2"),
         url("../../fonts/Montserrat-Regular.woff") format("woff");
    font-weight: normal;
    font-style: normal;
    font-display: swap;
  }
  
  @font-face {
    font-family: "Montserrat";
    src: url("../../fonts/Montserrat-Bold.woff2") format("woff2"),
         url("../../fonts/Montserrat-Bold.woff") format("woff");
    font-weight: bold;
    font-style: normal;
    font-display: swap;
  }

  • 16.05.2024 – Reset CSS und CSS-Variablen

    Implementierung von Reset CSS und CSS-Variablen. Wir haben zunächst ein Reset-CSS eingeführt, um eine konsistente Basis für alle Browser zu schaffen. Anschließend wurden CSS-Variablen definiert, um Farben, Schriftgrößen und Abstände zentral zu verwalten und eine einfache Wartung und Anpassung des Designs zu ermöglichen. 

    
  :root {
    /* Color */
    --black: #111111;
    --light-grey: #9e9b9b;
    --dark-grey: #1f1f1f;
    --grey: #555555;
    --white: #eee9e1;
    --purple: #a196df;
    --darkpurple: #392263;
    --yellowgreen: #90d491;
  
    /* Website - Colors */
    --bg-color: var(--black);
    --accent-color: var(--purple);
    --accent-color-2: var(--yellowgreen);
    --text-color: var(--white);
  
    /* Font-weight */
    --font-weight-light: 300;
    --font-weight-regular: 400;
    --font-weight-medium: 500;
    --font-weight-bold: 700;
  
    /* Font-family */
    --font-hanson: "Hanson", sans-serif;
    --font-montserrat: "Montserrat", sans-serif;
  
    /* Font-size */
    --font-size-h1: 2.5rem;
    --font-size-h2: 2rem;
    --font-size-h3: 1.5rem;
    --font-size-text-small: 0.9rem;
  
    --header-height: 70px;
    --footer-height: 182px;
  
    /* Header-animation */
    --top-size-small-1: -90px;
    --top-size-small-2: -180px;
    --top-size-big-1: -150px;
    --top-size-big-2: -300px;
  
    --line-height-heading: 1.2;
  
    --header-top-size-1: var(--top-size-small-1);
    --header-top-size-2: var(--top-size-small-2);
  
    @media screen and (min-width: 620px) {
      --header-top-size-1: var(--top-size-big-1);
      --header-top-size-2: var(--top-size-big-2);
    }
  }

  • 16.05.2024 – Erste Implementierung

    Erste Implementierung des Layouts und der Navigation. Wir haben die Grundstruktur der Webseite erstellt, inklusive Hauptnavigation, Header und Hauptinhaltsbereich. Die Entwicklung begann mit der mobilen Auflösung und wurde für größere Bildschirme erweitert. Dabei kam CSS Grid zum Einsatz, um eine flexible und anpassungsfähige Struktur zu gewährleisten.

  • 16.05.2024 – Umsetzung des Hauptinhalts

    Umsetzung des Hauptinhalts und der Projekte. Die Projekte wurden in einer Grid-Struktur angeordnet, um eine übersichtliche Darstellung zu gewährleisten. Dabei wurden die Projekte in einer oder mehreren Spalten angezeigt, abhängig von der Bildschirmgröße des Geräts.

  • 17.05.2024 – Implementierung des Footers

    Der Footer wurde in mehrere Abschnitte unterteilt, darunter soziale Medien, rechtliche Informationen und Kontaktinformationen. Dabei wurde Flexbox verwendet, um die Elemente zu positionieren und zu stylen. Die Listen wurden zentriert und mit einem Hover-Effekt versehen.

  • 18.05.2024 – Umsetzung der Funktion des Hamburger Menüs

    Die Funktion des Hamburger Menüs wurde mit JavaScript umgesetzt. Beim Klick auf das Hamburger Menü wird die Navigation eingeblendet und das Hamburger Menü wird animiert. Beim Klick auf einen Navigationslink wird das Hamburger Menü geschlossen. Mit der Escape-Taste kann das Hamburger Menü ebenfalls geschlossen werden.

  • 19.05.2024 – Die Über mich Seite

    Beim Meine Fähigkeiten Bereich auf der Über mich Seite wurde wieder Grid benutzt für das Grundlayout. Für die einzelnen Level wurde Flexbox benutzt. Die Seite wurde responsiv gestaltet.

  • 19.05.2024 – Kontaktformular

    Das Kontaktformular wurde mit Grid erstellt. Die einzelnen Felder wurden mit Flexbox gestaltet. Alle Inputs wurden mit Labels verknüpft. Das Aussehen von textarea und input wurde mit CSS angepasst. Die Seite wurde ebenfalls responsiv gestaltet.

  • 20.05.2024 – Testphase und Anpassungen

    In der Testphase wurden alle Funktionen und Inhalte überprüft und Anpassungen vorgenommen. Dabei wurde die Webseite auf verschiedenen Geräten und Browsern getestet, um eine optimale Darstellung und Funktionalität zu gewährleisten. Angefangen bei einer Auflösung von 320px bis 1920px. Die Webseite wurde auf Chrome, Firefox, Safari und Edge getestet. Dabei wurden die Inhalte und die Navigation überprüft und gegebenenfalls angepasst.

  • 20.05.2024 – Finalisierung

    Die Webseite wurde finalisiert und für die Abgabe vorbereitet. Dabei wurden die letzten Anpassungen vorgenommen, um die Performance zu optimieren. Die Webseite wurde bei einem Webhoster hochgeladen und getestet, um sicherzustellen, dass alles reibungslos funktioniert. Die JPG-Bilder wurden durch WebP-Bilder ersetzt, die zusätzlich komprimiert wurden. Die Webseite wurde auf Google PageSpeed getestet und mit dem W3C Validator überprüft. Die CSS-Datei wurde minifiziert. Es wurden die Dateien robots.txt und sitemap.xml erstellt.
